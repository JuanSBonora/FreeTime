SCHEME

SIMBOLOS Y SINTAXIS

Scheme posee una sintaxis reducida comparada a otros lenguajes. Utiliza la notación prefija. Por ejemplo, si se desea realizar la suma 1+1, se debe programar la siguiente instrucción

(+ 1 1)

SINTAXIS BASICA
Expresión -> átomo | lista átomo -> número | string | identificador | carácter | boolean lista -> '(' secuencia-expresión ')' secuencia-expresión -> expresión secuencia - expresión | expresión función -> ( nombre_función argumento1 argumento 2... argumento n)
